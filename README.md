### Clone Project Locally
```ssh
git clone git@github.com:nomasgrim/pwa-seed.git
```

### Move into root directory
```ssh
cd pwa-seed
```
### sync submodules
```ssh
git submodule update --init --remote
```

### Create .env
```ssh
cp pwa-server/.env.example pwa-server/.env
```

### Install Dependencies
```ssh
yarn
```

### Run Dev
```ssh
yarn run dev
```
